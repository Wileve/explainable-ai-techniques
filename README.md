Implementations and examples of various Explainable AI (XAI) methods like LIME and SHAP for interpreting black-box models.

## Features
- LIME (Local Interpretable Model-agnostic Explanations) for tabular data.
- SHAP (SHapley Additive exPlanations) for model interpretation.
- Examples with common machine learning models.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```python
python explainable_ai_techniques.py
```
